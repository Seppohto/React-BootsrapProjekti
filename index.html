Eteenpäin!

Arkkitehtuuri yhdistää Azuren sisällä useita palveluita ja tarjoaa lisäarvoa käsittelemällä avoimia API-rajanpintoja

Salaisuudet Keyvaultissa, Functiin APP funktiot tarjoillaa apimanagementin avulla eteenpäin. Data on tallessa Cosmos DB:ssä. Frontend ratkaisut tehty Flaskia ja Bootstrap templatea käyttäen, jonne on lisätty hieman javascriptiä, reactia ja tyylittelyjä. Toivottavasti nautitte!

  • 01 Web sivuja kahdella tapaa

    Rakensimme kaksi demoa Flaskille pythonilla ja tämän esityksen Bootstrap templatella ja lopussa olevan demon Reactilla.

  • 02 Api rajapinnoista vain Jsonia

    Halusimme varmistaa, että jokainen rajapinta tarjoaa mahdollisimman paljon käyttäjilleen ja siksi rajapinta on tehy jsonilla. Funktiot toimivat Azuren Function Appilla ja API Management tarjoaa ne käyttäjille.

  • 03 Tietoturva ja Laajennettavuus keskiössä

    Tietoturva oli läpi projektin jokaisen iteraation mielessä ja salaisuudet visusti keyvaultissa, eikä niitä säilytetä missään sen ulkopuolella. Laajennettavuutta tarjoaa Cosmos DB, jonne nyt tallennetaan koirien ja kissojen äänet. Ehkä jatkossa myös muiden eläinten.
